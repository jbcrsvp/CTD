# Linux Terminal Commend


## Vi editor command


vi : 실행

esc : 명령모드 진입

> 명령모드에서 
```
:q 종료
:q! 저장하지 않고 종료
:w 저장
:wq 저장후 종료
i  insert
a  append
h  왼쪽 이동
l 오른쪽 이동
j 아래로 이동
k 위로 이동
x del
X backspace
dd 현재 커서의 행 삭제
yy 현재 커서가 있는 행을 복사
p paste
/문자열 문자열 찾기
:set number 행 번호 표시
:%s/바꾸기전문자/바꿀문자  문자열 치환
```
