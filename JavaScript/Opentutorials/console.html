자바스크립트를 사용하기 위해 파일을 만들었는데
코드를 실행해야 하는 가벼운 사항이 있을 때,
웹 브라우저 상의 우클릭> 검사 > console로 들어감
console을 이용해서 자바스크립트 코드를 직접 웹 브라우저 상에서 파일을 만들지 않고 바로 실행할 수 있다.
.length
콘솔에서 실행시키는 js는 웹 페이지 안에 삽입되어 있는 것처럼 동작한다. 해당 웹 페이지를 대상으로 자바스크립트가 실행되게 된다.

활용하고자 하는 웹 페이지에서 우클릭> 검사
자스를 이용한다는 것은 이미 만들어져 있는 웹사이트를 나의 필요에 따라 맥락적으로 간단하게, 그러나 딱 맞는 코드를 작성해서 문제를 해결할 수 있다.
console을 활용하면 웹 페이지 안에서 자바스크립트 코드를 실행 할 수 있다.
