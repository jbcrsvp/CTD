Object

배열은 순서에 따라서 정리정돈 하는 기능 배열은 정보를 담는 그릇이며, 정보가 순서대로 저장 될 수 있음.
순서가 필요 없이 정보를 저장하는 방법이 객체(object)
객체는 '이름이 있는 정리정돈 상자' 정도로 이해하면 된다.
