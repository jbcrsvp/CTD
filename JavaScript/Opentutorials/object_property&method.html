(like coworkers)객체에는 함수, 데이터, 숫자 등 많은 것을 담을 수 있다.
If want to make Method in object, do "objectname.methodname"
coworkers.showAll = function(){ }은
function showAll(){ }과 같다.
객체.함수이름 = function(){}로 객체 안에 함수데이터를 담을 수 있다.

Property & Method

this를 잘 활용해야 한다.
in coworkers 라는 이름이 박혀있기 때문에, coworkers라는 객체의 이름이 바뀌게 되면,
데이터를 가져올 수 없다. 그래서 showAll이라는 함수 안에서 함수가 소속되어 있는 객체를 가리키는 약속된 기호인 this를 사용한다.

객체에 소속된 함수(여기서는 showAll)를 Method라고 하며
객체에 소속된 변수(여기서는 programmer, designer 등을 property라고 한다.
