## 자주쓰는 단축키
> Ctrl + Cmd+P : 모든 명령에 접근할 수 있는 명령 팔레트

> cmd + p : 파일이나 기호를 탐색

> cmd + \ : 에디터 나란히 놓고 편집

> cmd + 1,2,3 : 에디터(탭)간 이동

> cmd + shift + [ ] : 코드 접기 펴기

> cmd + shift + v : 마크다운 미리보기
