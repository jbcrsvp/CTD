combo box uses
tag'select' & tag'option'

<select name="변수명" size="1" multiple>
<option value="전달값-1" selected> 선택항목-1</option>
. . . </select>
.
: tag'select'은 콤보박스 입력을 나타내고, name은 웹 서버에 전달되는 데이터를 저장하는 변수명
size 속성은 화면에 표시되는 선택항목의 개수
multiple 속성 지정시 다중 선택 가능, 생략시 single
콤보박스 드랍다운 리스트에 표시되는 각 선택항목은 tag'option'으로
tag'option'의 value 속성은 해당 항목이 선택될 때 웹 서버에 전달할 값 지정
select 속성은 기본값 지정 , <selected>에서 multiple 속성을 지정하면 selected 속성은 여러 옵션 태그들에 지정할 수 있다.
